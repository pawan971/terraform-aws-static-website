Welcome to my AWS Static-Website deployed using Terraform pipeline

About This Project

This is a demo static website deployed on AWS using Terraform. The purpose of this project is to showcase how infrastructure as code can be used to easily deploy and manage web applications.

What I've Built

  • An S3 bucket configured for static website hosting
  • CloudFront distribution for content delivery
  • Route 53 configuration for custom domain (if applicable)
  • Automated deployment pipeline using Terraform

Technologies Used

  • Terraform
  • AWS S3
  • AWS Route 53 (optional)
  • HTML/CSS

This project demonstrates the power of Infrastructure as Code (IaC) and how it can streamline the process of setting up and managing cloud resources.
